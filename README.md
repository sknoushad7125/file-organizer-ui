# 📁 FileHub – Smart File Organizer UI

🌐 Live Demo:
https://file-organizer-ui.netlify.app

---

## 🚀 Overview

FileHub is a modern, browser-based file management system built using HTML, CSS, and Vanilla JavaScript. It simulates a real-world cloud storage interface similar to Google Drive, allowing users to manage files and folders dynamically without any backend.

The application is fully interactive and runs entirely in the browser using localStorage for data persistence.

---

## ✨ Features

### 📂 File & Folder Management
- Create, rename, and delete files/folders  
- Organized file display with icons  
- Context menu for quick actions  

### 🔍 Smart Search
- Real-time search filtering  
- Instant UI updates without reload  

### 🎨 UI & UX
- Grid & List view toggle  
- Clean modern UI with dark mode  
- Smooth hover interactions  
- Responsive layout  

### ⚙️ Advanced Interactions
- Right-click context menu  
- File preview modal  
- Action buttons (delete, rename, preview)  

### 💾 Persistence
- Data stored using localStorage  
- Retains state after refresh
